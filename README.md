### 🎯 安装
本项目已经npm发布了，可使用
```
npm install view-ui-components
```

### 🎯 使用

```
import viewUiComponents from 'view-ui-components' //引入库
import 'view-ui-components/lib/vui.css' // 引入样式
Vue.use(viewUiComponents)
```
